Pros

Pixels in comp match density independent units* in code. Comp can be reused as a starting point for other layouts in a multi-device project.

Cons

For Photoshop HiDPI/Retina users, canvas might be unusably small. Exported screens might not be pixel-perfect in prototypes.

Tips

Create assets at highest resolution possible to support all high density displays. Link assets as smart objects if using Photoshop.
