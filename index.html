Pros

Pixels in comp match density independent units* in code. Comp can be reused as a starting point for other designs in a multi-device system.

Cons

For Photoshop HiDPI/Retina users, canvas might be unusably small.

Tips

Create assets at highest resolution possible to support high density displays. Link assets as smart objects if using Photoshop. For this , export bitmap assets at × comp scale.

Pros

Ideal for designers: comfortable working size in Photoshop with HiDPI/Retina displays. Optimal for screen-based prototyping with After Effects or InVision.

Cons

Developer will need to divide all specs by to get density independent units* for code. Comp might be harder to reuse as a starting point for design on another device.

Tips

Append @x to the comp filename so that developers know to divide by for density independent units.

* 1 density independent unit = 1pt in iOS, 1dp in Android, 1px in web. Learn more about device metrics.
