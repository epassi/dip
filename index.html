This size makes it easier for developers to use your Photoshop, Sketch, or Illustrator source file as a measurement spec, because it matches the density independent (DIP) grid: an abstract pixel grid used by apps and browsers to approximate consistent sizing across displays of varying pixel densities.

×

Bitmap asset scale for
comp

More

Bitmap assets, including photos and other non-vector elements, should be cut for developers at times the size they appear in your comp. For example, a 100 × 100 image in your comp would need to be cut as a × asset. If you use Photoshop for your screen comp, it might be helpful to link your high resolution bitmap assets as smart objects.

More

If you use Photoshop with a Retina display and you find too small as a design space, use instead. This is also a practical comp size if you plan to do a lot of screen-based prototyping with tools like InVision. As a best practice, append @x to the PSD filename so that both designers and developers know they should divide by to get proper DIP measurements.
